# Penguins Database Analysis

This project provides a comprehensive analysis of the Palmer Penguins dataset. The analysis includes data exploration, visualization, statistical analysis, clustering, and machine learning models to understand the characteristics of different penguin species.

## Dataset

The Palmer Penguins dataset contains measurements of three penguin species: Adelie, Chinstrap, and Gentoo, collected from islands in the Palmer Archipelago, Antarctica.

## Features

- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Statistical tests
- Data visualization
- Clustering analysis (K-Means, DBSCAN, Hierarchical)
- Dimensionality reduction (PCA)
- Model evaluation

## Requirements

To run this project, you need Python 3.7+ and the following packages:

- numpy
- pandas
- seaborn
- matplotlib
- scipy
- scikit-learn

Install the dependencies using:

```bash
pip install -r requirements.txt
```

## Usage

1. Clone or download this repository.
2. Install the required packages.
3. Open the `penguinsDatabaseFinal.ipynb` notebook in Jupyter Notebook or JupyterLab.
4. Run the cells in order to execute the analysis.

## Files

- `penguinsDatabaseFinal.ipynb`: The main Jupyter notebook containing the analysis.
- `requirements.txt`: List of Python dependencies.
- `.gitignore`: Git ignore file for Python projects.
